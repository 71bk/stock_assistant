## OCR Cleanup 設定
- APP_OCR_CLEANUP_ENABLED: 是否啟用 SUPERSEDED 清理（預設 false）
- APP_OCR_CLEANUP_SUPERSEDED_DAYS: 保留天數（預設 30）
- APP_OCR_CLEANUP_CRON: 清理排程（預設 0 0 3 * * *）
- APP_OCR_CLEANUP_BATCH_SIZE: 每批刪除筆數（預設 200）

## OCR Queue 設定
- APP_OCR_QUEUE_WORKER_CONCURRENCY: OCR worker 併發數（預設 2）
- APP_OCR_QUEUE_WORKER_QUEUE_CAPACITY: OCR worker 佇列容量（預設 50）

## Auth Rate Limit
- APP_AUTH_REFRESH_RATE_LIMIT: Refresh endpoint 每個 IP 的限制次數（預設 30）
- APP_AUTH_REFRESH_RATE_WINDOW: Refresh rate limit 視窗（預設 60s）
