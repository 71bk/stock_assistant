# 部署與基礎設施

> 狀態：已整理可落地版本 (2026-02-09)

## Docker Compose

- 檔案位置：`apps/backend/invest-assistant-backend/compose.yaml`
- 目前內建服務：
  - `postgres`：`pgvector/pgvector:pg16`
  - `redis`：`redis:7.2`（AOF 開啟）
- 啟動：
  - `cd apps/backend/invest-assistant-backend`
  - `docker compose up -d`
- 停止：
  - `docker compose down`

## Nginx 反向代理（含 SSE）

- 建議反代 `/api` 到 Spring Boot（context path 已是 `/api`）。
- SSE 必要設定：
  - `proxy_buffering off`
  - `proxy_read_timeout` 調大（例如 300s）
  - `chunked_transfer_encoding on`

## 環境變數與 Secrets

- 核心：
  - `DB_HOST` `DB_PORT` `DB_NAME` `DB_USER` `DB_PASSWORD`
  - `REDIS_HOST` `REDIS_PORT`
  - `APP_AUTH_JWT_SECRET`
- OAuth：
  - `SPRING_SECURITY_OAUTH2_CLIENT_REGISTRATION_GOOGLE_CLIENT_ID`
  - `SPRING_SECURITY_OAUTH2_CLIENT_REGISTRATION_GOOGLE_CLIENT_SECRET`
- 檔案：
  - `APP_FILES_PROVIDER`（`local` / `s3` / `minio`）
  - `APP_FILES_LOCAL_PATH` / `APP_FILES_BUCKET` / `APP_FILES_ENDPOINT`
- OCR Queue：
  - `APP_OCR_QUEUE_WORKER_CONCURRENCY`（預設 2）
  - `APP_OCR_QUEUE_WORKER_QUEUE_CAPACITY`（預設 50）
- OCR Cleanup：
  - `APP_OCR_CLEANUP_ENABLED`（預設 false）
  - `APP_OCR_CLEANUP_SUPERSEDED_DAYS`（預設 30）
  - `APP_OCR_CLEANUP_CRON`（預設 `0 0 3 * * *`）
  - `APP_OCR_CLEANUP_BATCH_SIZE`（預設 200）
- Auth Rate Limit：
  - `APP_AUTH_REFRESH_RATE_LIMIT`（預設 30）
  - `APP_AUTH_REFRESH_RATE_WINDOW`（預設 60s）
- Stocks Rate Limit：
  - `stock.rate-limit.enabled`（預設 true）
  - `stock.rate-limit.window-ms`（預設 1000）
  - `stock.rate-limit.alpaca-limit`（預設 8）
  - `stock.rate-limit.fugle-limit`（預設 8）
- Observability：
  - Actuator 已開：`/actuator/health`、`/actuator/metrics`、`/actuator/prometheus`
- Sentry：
  - Runtime 上報使用 `sentry.dsn`
  - Build source bundle upload 僅在 `SENTRY_AUTH_TOKEN` 存在時啟用（profile: `sentry-source-upload`）

## CI/CD（GitHub Actions）

- 建議最小流程：
  - `mvn -pl modules/app-api -am test-compile`
  - 架構規則測試（ArchUnit）
  - `mvn -pl modules/app-stocks -am test`
  - 建置 image 並推送 registry
  - 目標機器 `docker compose pull && docker compose up -d`
- 生產環境若要上傳 Sentry source bundle，CI 要注入 `SENTRY_AUTH_TOKEN`。

## 監控與備份

- Metrics/Monitoring：
  - Prometheus 抓取 `/actuator/prometheus`
  - 監控重點：`stock.external.*`、`stock.cache.requests`、`stock.singleflight.joined`
- Logs：
  - 保留 `traceId` 以便 API 請求追蹤
  - 外部行情呼叫保留 `vendor/endpoint/status/latency_ms/request_id/trace_id`
- 備份建議：
  - PostgreSQL：每日 dump + 7/30 天分層保留
  - Redis：AOF + 定期快照（若含重要 session 狀態）
